Vendedor de bilhetes de loteria:
Veja como jogar na loteria:
Selecione cinco números entre 1 e 20. Seu prêmio é determinado pela quantidade de números vencedores que você acertar.
O primeiro lugar vai para aqueles que acertarem todos os cinco números vencedores. O valor do prêmio do primeiro lugar é %prize5% % do valor total do jackpot, menos o prêmio do quarto lugar.
O segundo lugar vai para aqueles que acertarem quatro dos cinco números vencedores. O valor do prêmio do segundo lugar é %prize4% % do valor total do jackpot, menos o prêmio do quarto lugar.
O terceiro lugar vai para aqueles que acertarem três dos cinco números vencedores. O valor do prêmio do terceiro lugar é %prize3% % do valor total do jackpot, menos o prêmio do quarto lugar.
Aqueles que escolherem dois ou menos números vencedores poderão receber a adena %prize2%.
O prêmio em dinheiro para cada categoria será dividido igualmente entre os vencedores. Se não houver ganhadores em nenhuma categoria, esses prêmios serão transferidos para o próximo jackpot.
Os bilhetes de loteria são vendidos semanalmente, começando às 19h10 de cada sábado até as 18h50 do sábado seguinte.
Os números vencedores são anunciados às 19h00 todos os sábados. Todos os horários estão sujeitos a alterações.
Desejo-lhe sorte! Aproveite a loteria.
